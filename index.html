Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • !DOCTYPE html declaration tells the browser what document type it should expect when it is opened.
  • lang="eng" in the head element tells the browser the language for the webpage.
  • HTML attributes are always defined in the beginning element, never the closing element.
  • Search engine optimization (SEO) is the process of improving the ranking of a website in search engines.
  • The title element defines the title for a webpage.
  • The style element defines the CSS styles associated with a website.
  • The base element defines the base URL for a webpage.
  • The link element connects an external resource to the HTML document.
  • The meta element defines metadata such as the character set, description, keywords, auther, and viewport.
  • The script element embeds client side scripts in the HTML document.

Git

  • The ssh-agent is a program that keeps track of user's identity keys and passphrases.
  • The git config command lets you change the email address you assosiate with your Git commits.
  • An SHA, or hash, is actually a long string of numbers or letters, like this:"e0eaed45609ae40ada29664384e679ada5d5004a"However, when using the SHA to access a version of their code, developers often type out only the first seven numbers or letters. This gives just the right amount of information to allow the right version to be retrieved and eliminates the need to copy the entire SHA.
  • The git revert command reverts to a previous version of the workflow.
  • Git Commands
  • git status: checks what branch we are currently
  • git checkout -b branch-name: creates a new branch and switches to it
